# CastDrop ⚡ by Talons Protocol

Pick a vibe. Get a cast with image. Share to Warpcast in one tap.

## Features
- 12 categories: GM, Crypto, Motivation, Tech, Humor, Alpha, DeFi, NFT, Farcaster, Builder, AI, Trading
- 25+ unique casts per category
- Real topic-related images (Unsplash)
- One-tap Warpcast share with embedded image
- Cast history log
- Dark cyberpunk UI

## Deploy
1. Push to GitHub
2. Import on Vercel
3. Deploy — no env vars needed

## Stack
- Next.js 15 / React 19
- Tailwind CSS 4
- Unsplash images (no API key)
- Warpcast compose URL with `embeds[]`
